## 📖 Project Overview

This repository contains the code and analytical pipeline for my graduation thesis. The project focuses on conducting **Exploratory Data Analysis (EDA)**, **Data Visualization**, and applying **Machine Learning** models to the Divvy Bike Sharing system dataset from Chicago.

The main objectives of this thesis are:
1. **Understand Rider Behavior:** Analyze the differences between annual members and casual riders.
2. **Temporal Trends:** Uncover daily, weekly, and monthly seasonality and usage patterns.
3. **Spatial Analysis:** Identify the most popular stations and high-demand areas.
4. **Predictive Modeling:** Build machine learning models to forecast trip durations and predict high-demand nodes *(Upcoming Phase)*.

## 🚀 Getting Started

### 1. Prerequisites
Make sure you have Python 3.8+ installed. It's recommended to run this inside a virtual environment.

```bash
# Create a virtual environment
python -m venv .venv
source .venv/Scripts/activate  # On Windows

# Install the required dependencies
pip install -r bike_thesis/requirements.txt
```

### 2. Data Preparation
To run the analysis, you need the Divvy dataset.
1. Download the trip data CSV files for the desired year (e.g., 2025) from the [Divvy Data portal](https://divvybikes.com/system-data).
2. Place these CSVs inside the root `2025/` folder.

### 3. Running the Pipeline
Navigate to the `bike_thesis` directory and run `main.py` as a module. This orchestrates loading, cleaning, transforming, and visualizing the data.

```bash
cd bike_thesis
python -m src.main
```

The script will:
- Process the `.csv` files and cache cleaned data in `data/processed/`.
- Generate over 8 different visualizations (e.g., *Heatmaps, Daily/Monthly Distributions, Member vs Casual stats*) and save them to `outputs/figures/`.
- Print descriptive statistics to the console.

## 📈 Key Visualizations Automated
The pipeline automatically generates several aesthetic, thesis-ready plots using `seaborn` and `matplotlib`:
* **Ride Demand Heatmap:** Weekday vs Hour usage intensity.
* **Monthly & Daily Ride Counts:** With 7-day moving averages for trend following.
* **Top 10 Start Stations:** Barcharts highlighting system bottlenecks.
* **Ride Duration Distribution:** Analyzing trip efficiency and length limits.
